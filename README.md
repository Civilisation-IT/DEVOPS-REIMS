# TP1 DevOps - Provisioning et Configuration

Ce dépôt contient les ressources du TP1 pour :
- provisionner l'infrastructure avec Terraform/OpenTofu ;
- configurer les machines avec Ansible.

## Guide d'exécution

Le déroulé complet des étapes est documenté dans :
- [step.md](./tp1/step.md)

Commencez par ce fichier pour suivre l'ordre des chapitres :
1. Setup
2. Terraform
3. Ansible

## Structure du projet

- `terraform/` : code d'infrastructure (Terraform/OpenTofu)
- `ansible/` : playbooks, rôles et inventaire Ansible
- `step.md` : instructions pas à pas
